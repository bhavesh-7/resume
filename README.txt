Resume Template
===============

This folder contains a LaTeX resume template (resume.tex) and its compiled PDF (resume.pdf). Follow the instructions below to edit and preview the .tex file.

How to Edit and Preview
-----------------------
1. Web:
   Use Overleaf (https://overleaf.com):
   - Create a free account.
   - Upload the resume.tex file.
   - Edit and preview directly in your browser.

2. Windows:
   Use TeXworks (https://www.tug.org/texworks/) or MiKTeX (https://miktex.org):
   - Install the editor.
   - Open resume.tex in the editor.
   - Compile using PDFLaTeX.

3. Linux:
   Use TeXStudio (https://texstudio.org/):
   - Install TeXStudio (e.g., `sudo dnf install texstudio` on Fedora).
   - Open resume.tex in TeXStudio.
   - Compile and preview the PDF.

Files Included
--------------
- resume.tex: The LaTeX source file.
- resume.pdf: The compiled PDF file.
